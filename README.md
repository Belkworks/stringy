
# Stringy
*A pretty printer for Synapse.*

**Importing with [Neon](https://github.com/Belkworks/NEON)**:
```lua
Stringy = NEON:github('belkworks', 'stringy')
```

## Usage
Call **Stringy** to pretty print all passed parameters.  
**Stringy**: `Stringy(...) -> nil`
````lua
Stringy('hello', {world = true})
```
